## 이후 고려할 것
- 입력에 원하는 정보 말고 불필요한 정보가 많이 들어있을 때 (책을 추천해줘. 라고 할 때, 책을 추천해줘 까지도 유사한 문장을 비교하면 어떡하지?)
- 유사도가 너무 작을 때.. 임계값을 정해야 (top 5 내에 0.3이 있으면?)
